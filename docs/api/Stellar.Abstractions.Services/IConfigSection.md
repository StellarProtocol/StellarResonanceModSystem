# IConfigSection interface

A named subset of a plugin's configuration. Read access via [`Get`](./IConfigSection/Get.md) is lock-free against the in-memory cache; writes via [`Set`](./IConfigSection/Set.md) are cached and persisted to disk on [`Save`](./IConfigSection/Save.md). Supported `T`: primitives (Int32, Int64, Boolean, String, Single, Double), arrays of primitives, and string-keyed dictionaries of primitives. Records and complex objects are out of scope for v1 — use multiple keys instead.
