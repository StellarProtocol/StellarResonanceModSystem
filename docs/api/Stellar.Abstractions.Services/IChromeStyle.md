# IChromeStyle interface

User-selected global chrome control styles — the button + scrollbar look applied to every GlassMenu panel. Persisted with the theme; changing either fires the theme's [`ActiveChanged`](./INamedTheme/ActiveChanged.md) so the renderer rebuilds its skin. A plugin may pin its own per-window button and scrollbar style via [`ButtonElement`](./ButtonElement.md), which the user cannot override here.
