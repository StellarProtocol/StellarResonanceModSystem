# ILauncher interface

Registry of launcher entries. Plugins call [`Register`](./ILauncher/Register.md) to add a tile to the Stellar launcher menu and hold the returned IDisposable to remove it on unload. [`Entries`](./ILauncher/Entries.md) preserves registration order.
