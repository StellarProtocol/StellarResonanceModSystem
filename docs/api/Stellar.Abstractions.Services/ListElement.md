# ListElement record

Variable-length list, bounded by Slots.Count. All slots built once; the first *VisibleCount*() are SetActive-shown each refresh. *Columns*&gt;1 grids them. *CellWidth*/*CellHeight* (when &gt; 0, multi-column only) override the grid's default cell size — use to widen columns past the framework default (e.g. the StatInspector mini-HUD, whose icon+label+value row needs more than the default cell width).
