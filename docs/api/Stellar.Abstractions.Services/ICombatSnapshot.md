# ICombatSnapshot interface

Polled snapshot of the local player's combat state plus a ring buffer of recently observed events. Mirrors the polled half of the original mixed combat surface; the event stream lives on [`ICombatEvents`](./ICombatEvents.md) and per-entity lookups live on [`ICombatLookup`](./ICombatLookup.md). All reads are safe from the Unity main thread.
