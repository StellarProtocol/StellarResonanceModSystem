# IPlayerStats interface

Read-only access to live player attribute values. Each tracked attribute is identified by its `Zproto.EAttrType` integer code (e.g. 11011 for `AttrStrengthTotal`). Pair with [`GetAttribute`](./IGameDataCombat/GetAttribute.md) to resolve the localized label and [`GetAttributeProfile`](./IGameDataCombat/GetAttributeProfile.md) to discover the UI group. Attributes must be [`Subscribe`](./IPlayerStats/Subscribe.md)d before they are sampled. The probe polls only subscribed IDs each `Game.Update` tick — sampling all 1289 EAttrType members per frame would be wasteful. Unsubscribed IDs return null.
