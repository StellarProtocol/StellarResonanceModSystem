# IColorSlot interface

Handle returned by [`IColorRegistry`](./IColorRegistry.md)'s `Register` methods. Read [`Value`](./IColorSlot/Value.md) when drawing — it resolves to the colour for the active theme, honouring user overrides. Cache the handle, not the value.

Disposing a slot unregisters it from the owning [`IColorRegistry`](./IColorRegistry.md).
