# ICombatEvents interface

Combat event stream. Mirrors the event half of the original mixed combat surface; polled state lives on [`ICombatSnapshot`](./ICombatSnapshot.md) and per-entity lookups on [`ICombatLookup`](./ICombatLookup.md). Events always fire on the Unity main thread (drained once per `Game.Update` postfix).
