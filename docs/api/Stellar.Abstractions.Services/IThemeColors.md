# IThemeColors interface

Theme colour facade — zero declared members. Plugins access the full colour vocabulary (base, HUD, menu) through this single type; the split into [`IThemeBaseColors`](./IThemeBaseColors.md) + [`IThemeHudColors`](./IThemeHudColors.md) + [`IThemeMenuColors`](./IThemeMenuColors.md) exists so each facet stays inside the STELLAR0005 8-member cap (the analyzer counts declared members per interface; inherited members are unbounded).
