# IEntityDetail interface

Per-entity detail captured from the combat wire's `SyncNearEntities` attribute stream — the full broadcast attribute map and equipment for any in-AOI entity. Identity/skills come from [`ICombatLookup`](./ICombatLookup.md); derived self-only stats from [`IPlayerStats`](./IPlayerStats.md).
