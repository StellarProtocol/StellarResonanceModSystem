# IResonanceState interface

Read-only view of the local player's equipped Battle Imagines (Resonance Skills), decoded from `CharSerialize.resonance` (wire field 28). The equipped set drives the CombatMeter's trailing Imagine icons for the self row. Populated on the network receive thread and read on the main thread; implementations publish an immutable snapshot so reads are lock-free.
