# IGameAssets interface

Toolkit service for loading game assets (textures, sprites) via the game's own Addressables loader (`ZResLoader`). Provides an opaque texture handle so plugins can render icons without a direct Unity dependency on `Texture2D`.

## Remarks

Loading is asynchronous (driven by the game's UniTask Addressables pipeline). Call [`LoadProfessionIcon`](./IGameAssets/LoadProfessionIcon.md) every frame from an Update tick; it returns `null` while the load is in progress and the resolved texture handle once the load succeeds. A permanently failed load also returns `null` — the caller should fall back to text-only rendering.
