# IProfileCardActions interface

Plugin-facing service for contributing buttons to the game's native profile card action bar. Each registered [`ProfileCardActionSpec`](./ProfileCardActionSpec.md) becomes a styled button the framework injects into the card on open; the returned handle removes the action when disposed.
