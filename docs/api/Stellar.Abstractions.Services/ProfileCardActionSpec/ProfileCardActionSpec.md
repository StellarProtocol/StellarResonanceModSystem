# ProfileCardActionSpec constructor

A button a plugin contributes to the game's native profile card action bar. The framework injects a styled button per registered spec into the card's action row (icon over label, matching the native action buttons) and, on click, resolves the carded player and invokes *OnClick* with their [`EntityId`](../../Stellar.Abstractions.Domain/EntityId.md).

```csharp
public ProfileCardActionSpec(string Id, string Label, byte[]? IconPng, Action<EntityId> OnClick)
```

| parameter | description |
| --- | --- |
| Id | Stable identifier for the action; used to name the injected button and to dedupe registrations. Must be unique among registered actions. |
| Label | Text shown below the icon on the injected button. |
| IconPng | Raw PNG bytes for the button icon (same Unity-free convention as [`IconPng`](../MenuButtonSpec/IconPng.md)); the framework rasterises it to a texture. Null renders a label-only button. |
| OnClick | Invoked on the main thread with the carded player's [`EntityId`](../../Stellar.Abstractions.Domain/EntityId.md) when the button is clicked. |
