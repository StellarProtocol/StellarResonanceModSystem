# CooldownTileElement constructor

One cooldown/debuff tile for the CooldownBar: a fixed icon square (game-asset art via *Icon*/*Uv*) with an *Accent*-tinted outline, a foot fill-bar (width = *Fill01*, 0..1), an optional ★ corner badge when *IsImagine*, an optional ×N charge count when *ChargeCount* &gt; 1, and a centred *Seconds* caption below — all poll-diffed each refresh. The framework binds the boxed texture on the window's refresh pass (renders without the runtime ticker, unlike [`GameTextureElement`](../GameTextureElement.md)), so it appears immediately. Compose pooled tiles inside a [`RowElement`](../RowElement.md) wrapped in [`ConditionalElement`](../ConditionalElement.md)s for active-only collapse.

```csharp
public CooldownTileElement(Func<object?> Icon, Func<UvRect> Uv, Func<float> Fill01, 
    Func<string> Seconds, Func<ColorRgba> Accent, Func<bool> IsImagine, Func<int> ChargeCount)
```

| parameter | description |
| --- | --- |
| Icon | Supplies the boxed `UnityEngine.Texture` for the tile art (null = neutral box). |
| Uv | The atlas UV sub-rect for the art (0..1, bottom-left origin). |
| Fill01 | Foot fill-bar fraction (0..1) — typically cooldown/​debuff completion. |
| Seconds | Remaining-time caption shown below the icon. |
| Accent | Tile accent — outline + fill + caption tint (e.g. cyan = cooldown, red = debuff). |
| IsImagine | When true, a ★ corner badge marks an Imagine-lockout tile. |
| ChargeCount | When &gt; 1, a ×N badge shows remaining charges. |
