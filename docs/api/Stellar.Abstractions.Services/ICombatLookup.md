# ICombatLookup interface

Per-entity combat lookups. Covers any entity observed in AOI on the combat wire — not limited to party members or the local player. Mirrors the lookup half of the original mixed combat surface; the polled local snapshot lives on [`ICombatSnapshot`](./ICombatSnapshot.md) and the event stream lives on [`ICombatEvents`](./ICombatEvents.md).
