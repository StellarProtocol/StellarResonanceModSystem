# ICombatLookup.GetLiveHps method

5-second sliding-window healing-per-second for a heal source ([`SourceId`](../../Stellar.Abstractions.Domain/CombatEvent.DamageDealt/SourceId.md) with `IsHeal=true`). Returns 0 for entities that have never been observed as a heal source, or when the source's last heal fell out of the 5-second window. Aggregated automatically as [`DamageDealt`](../../Stellar.Abstractions.Domain/CombatEvent.DamageDealt.md) heal events flow through — no per-plugin tracking needed.
