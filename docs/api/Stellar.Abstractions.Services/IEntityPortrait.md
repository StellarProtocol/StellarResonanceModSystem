# IEntityPortrait interface

Live, posed 3D portrait of a player entity — full body, real outfit, idle-animated — rendered by the game's own UI-model pipeline into a render texture that a [`RenderTextureHostElement`](./RenderTextureHostElement.md) displays. Works for the local player and any other player (the subject is created from the player's social data, the same way the game's own character preview windows do it). The plugin calls [`Show`](./IEntityPortrait/Show.md) when its portrait slot opens on an entity and reads [`Texture`](./IEntityPortrait/Texture.md) (the boxed `UnityEngine.Texture`) to display it.
