## Remarks

Phase 9a supplement renamed Mana → Stamina to match Star Resonance's own terminology. The underlying probe (`PandaPlayerStateProbe`) already reads the game's `LuaOriginEnergy` / `LuaMaxOriEnergy` fields, which are the stamina pool — the rename is only the C# identifier on our side.
