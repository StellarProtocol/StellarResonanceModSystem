# TextAlign enumeration

Horizontal text alignment within the text's cell. Default Left; Right is used for numeric table columns (so magnitudes line up against the right edge of a fixed-width [`CellElement`](./CellElement.md)).

```csharp
public enum TextAlign
```

## Values

| name | value | description |
| --- | --- | --- |
| Left | `0` | Align text to the left of the cell. |
| Center | `1` | Centre text horizontally in the cell. |
| Right | `2` | Align text to the right of the cell (use for numeric columns). |
