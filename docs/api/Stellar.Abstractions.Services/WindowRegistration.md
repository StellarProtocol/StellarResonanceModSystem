# WindowRegistration record

A registered interactive window: its spec + the root element to render. *TitleLeading* (when set) replaces the title-bar text (e.g. a logo + wordmark row); *TitleTrailing* is built right-aligned before the ✕ (e.g. mode/rotate toggles). Both null ⇒ the chrome draws the plain Title text. *OnClose* (when set) is invoked by the ✕ instead of just hiding the GameObject — wire it to the window's [`SetVisible`](./IWindowControl/SetVisible.md)(false) so [`IsShown`](./IWindowControl/IsShown.md) stays in sync (otherwise a rail/hotkey toggle needs two presses to reopen after a ✕ close).
