# WindowSpec constructor

Immutable declaration of a plugin window's identity, initial geometry, and chrome options. Passed to [`Register`](../../Stellar.Abstractions.Services/IWindowHost/Register.md) to create the window.

```csharp
public WindowSpec(string Id, string Title, WindowRect DefaultRect, WindowCategory Category, 
    WindowPanelStyle Style)
```

| parameter | description |
| --- | --- |
| Id | Stable string id, unique per plugin. Used to persist position and hotkey binding. |
| Title | Display title shown in the title bar and Settings layout editor. |
| DefaultRect | Initial position and size applied on first run (before user adjustments are persisted). |
| Category | Logical category that determines which group this window appears in within the layout editor. |
| Style | Visual chrome style applied to the window frame. |
