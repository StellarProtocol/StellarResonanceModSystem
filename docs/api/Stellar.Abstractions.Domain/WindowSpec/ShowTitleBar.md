# WindowSpec.ShowTitleBar property

GlassMenu only: draw the top title bar. Defaults true. Set false for windows that self-compose their own header inside the body (the launcher, whose header is top in Full/vertical but a LEFT strip in horizontal — a single fixed top bar can't express both). With no title bar the whole frame becomes the drag handle (if [`Draggable`](./Draggable.md)), and the body must supply its own close affordance.
