# EnchantItemInfo constructor

One `EquipEnchantItemTable` row — a socketed gem at a specific level. Resolved from the wire's `(enchant_item_type_id, enchant_level)` pair: the wire level is an internal progression index, NOT the displayed level — the displayed level is baked into the gem ITEM's name ([`GemItemId`](./GemItemId.md) → item-table name, e.g. "Crimson Foxen Sigil Lv.2").

```csharp
public EnchantItemInfo(int GemItemId, IReadOnlyList<EnchantEffect> Effects)
```

| parameter | description |
| --- | --- |
| GemItemId | Item id of the gem at this level; its item-table name carries the display level. |
| Effects | Flat stat grants of the gem (attr id + value), e.g. Haste +560. Never null. |
