# EnchantItemInfo structure

One `EquipEnchantItemTable` row — a socketed gem at a specific level. Resolved from the wire's `(enchant_item_type_id, enchant_level)` pair: the wire level is an internal progression index, NOT the displayed level — the displayed level is baked into the gem ITEM's name ([`GemItemId`](./EnchantItemInfo/GemItemId.md) → item-table name, e.g. "Crimson Foxen Sigil Lv.2").

public struct EnchantItemInfo : IEquatable<EnchantItemInfo>
