# EquipResult enumeration

Outcome of an [`IModuleEquip`](../Stellar.Abstractions.Services/IModuleEquip.md) dispatch. Maps the game's `EErrorCode` reply to plugin-friendly cases — plugins should never need to read raw error codes.
