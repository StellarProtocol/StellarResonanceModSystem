# GearAttrRolls constructor

The four rolled-attribute groups of an owned gear piece (wire `EquipAttr` maps `basic_attr`, `advance_attr`, `recast_attr`, `rare_quality_attr`). Lists are never null; empty groups use shared empty arrays.

```csharp
public GearAttrRolls(IReadOnlyList<GearAttrRoll> Basic, IReadOnlyList<GearAttrRoll> Advanced, 
    IReadOnlyList<GearAttrRoll> Recast, IReadOnlyList<GearAttrRoll> Rare)
```

| parameter | description |
| --- | --- |
| Basic | Fixed basic attributes (white lines). |
| Advanced | Rolled advanced attributes (green lines). |
| Recast | Reforged/recast attributes (yellow lines). |
| Rare | Rare-quality attributes (purple lines). |
